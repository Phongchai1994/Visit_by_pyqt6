# Prison Visit Management System

ระบบจัดการการเยี่ยมผู้ต้องขังในเรือนจำ

## คุณสมบัติหลัก
- ระบบล็อกอินผู้ใช้ (PyQt6)
- เชื่อมต่อฐานข้อมูล PostgreSQL
- เมนูหลักแยกซ้าย-ขวา (MenuFrame)
- รองรับการจัดการผู้ต้องขัง การเยี่ยม รายงาน ฯลฯ

## การติดตั้ง
1. สร้าง virtual environment (แนะนำ)
2. ติดตั้ง dependencies:
   ```
   pip install -r requirements.txt
   ```
3. สร้างไฟล์ `.env` และกำหนดค่าการเชื่อมต่อฐานข้อมูล เช่น
   ```
   PG_HOST=... 
   PG_PORT=...
   PG_NAME=...
   PG_USER=...
   PG_PASS=...
   ```

## การใช้งาน
- รันโปรแกรมด้วย
  ```
  python main.py
  ```
- ระบบจะเปิดหน้าต่างล็อกอิน หากสำเร็จจะเข้าสู่เมนูหลัก

## หมายเหตุ
- **ห้าม push ไฟล์ .env ขึ้น GitHub**
- สามารถปรับแต่งเมนูและหน้าต่างหลักได้ใน `ui/menu_frame.py` และ `ui/main_app.py`
